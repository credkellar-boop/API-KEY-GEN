# Security Policy
## Supported Versions
Currently, only the latest version of API-KEY-GEN receives security updates.

| Version | Supported |
| :--- | :--- |
| 1.0.x | :white_check_mark: |
| < 1.0 | :x: |

## Reporting a Vulnerability
If you discover a security vulnerability within API-KEY-GEN, please **do not** open a public issue. 
Instead, please send an email to `security@yourdomain.com`. All security vulnerabilities will be promptly addressed. We aim to acknowledge reports within 48 hours and provide a timeline for a patch.
